# docs/concepts/ — future-product and downstream-leaf concepts

This directory is the home for **concepts that are not yet plans**: future
products, downstream-leaf ideas, and design directions that have been worked out
far enough to be worth preserving, but that are deliberately *not* part of this
repo's plan sequence. A concept lands here so the thinking survives — an account
migration, a context reset, a long gap — without being mistaken for active work.

**A concept here is not-a-plan.** It is not numbered in any repo's Plans
Registry, it is not on the active task list, and it is not started until it is
explicitly *promoted* to a plan (each concept names its own promotion gate). The
plan that *lands* a concept document is numbered; the concept it documents is
not.
